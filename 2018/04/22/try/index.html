Question 1

Using basic statistical properties of the variance, as well as single-variable calculus, derive (5.6). In other words, prove that $\alpha$ given by (5.6) does indeed minimize \(Var({\alpha}X+(1−\alpha)Y)\).

We know from the statistical properties of the variance that:
$$Var({\alpha}X+(1−\alpha)Y)=\alpha^2\sigma_X^2+2\alpha(1-\alpha)\sigma_{XY}+(1-\alpha)^2\sigma_Y^2$$
In order to find the turning point of this function, let the first derivative of the right hand side of the above equation relative to $\alpha$ equals to zero:
$$\frac{\partial}{\partial\alpha}Var({\alpha}X+(1−\alpha)Y)=2\alpha\sigma_X^2+2\sigma_{XY}-4\alpha\sigma_{XY}-2\sigma_Y^2+2\alpha\sigma_Y^2=0$$
which is equivalent to:
$$\alpha=\frac{\sigma_Y^2-\sigma_{XY}}{\sigma_X^2+\sigma_Y^2-2\sigma_{XY}}$$
Now we can check whether the alpha value at this point is the minimum, which is equivalent to check whether the second derivative of $Var({\alpha}X(1−\alpha)Y)$ is positive:
$$\frac{\partial^2}{\partial\alpha^2}Var({\alpha}X+(1−\alpha)Y)=2\sigma_X^2+2\sigma_Y^2-4\sigma_{XY}=2(\sigma_X-\sigma_Y)^2\ge0$$

With the above deduction process we can prove that $\alpha$ given by (5.6) does indeed minimize $Var({\alpha}X+(1−\alpha)Y)$

Question 3

We now review k-fold cross-validation.

(a) Explain how k-fold cross-validation is implemented.

The k-fold cross-validation involves randomly dividing the set of observations into k groups, or folds, of approximately equal size. Then, we treat the first fold as validation dataset, and train the model on the remaining k-1 foldss. After that, we compute the mean squared error ($MSE_1$) on the observations in the validation set.This procedure is then repeated k times. For each time, a diﬀerent group of observations will be treated as a validation set. This process results in k estimates of the test error, $MSE_1$, $MSE_2$,…, $MSE_k$. The k-fold cross-validation estimate is computed by averaging these values:

$$CV_{(k)}=\frac{1}{k}\sum_{i=1}^kMSE_i$$

(b) What are the advantages and disadvantages of k-fold cross-validation relative to:

i. The validation set approach?

The validation set approach has two disadvatages when comparing to the k-fold cross-validation method:

Firstly, k-fold cross validation can reduce the high variability in test error rate when using the validation set approach by performing the validation multiple times with different folds of data to reduce the randomness in selecting training and testing dataset.

Secondly, k-fold cross validation also reduce the bias when using the validation set approach, in which the training set is typically around half the size of the original dataset. In contrast, instead of only fitting the data with one subset of data, k-fold cross validation repeatedly fit the data k times on different subsets of the original dataset, and thus reduce the probability of overestimating the test error.
